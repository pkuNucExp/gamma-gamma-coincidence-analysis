## 命令

- nc/newcanvas ==> new canvas
- sr/setrange ==> set range
- gm ==> gate multi
- tpj/totalproject ==> total project
- snp ==> set number of peaks
- spw ==> set peaks width
- spt ==> set peaks threshold
